The project is structured as a Monorepo:

- **`apps/web`**: Next.js 14+ Application (Frontend)
    - Host Dashboard (Monitor interviews, view reports)
    - Candidate Room (Silent Proctoring, WebRTC, Fullscreen enforcement)
- **`apps/api`**: NestJS Application (Backend)
    - WebSocket Gateway (Signaling, Alerts)
    - REST API (Auth, Interview Management)
    - Worker Queues (Processing recordings)
